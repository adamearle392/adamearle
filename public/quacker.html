<< Quacker </> Farm Based Social Media Android App


Why?

For a laugh and a learning experience. I wanted to try and fully implement a serverless solution like Firebase

In previous Android Apps I have built, I have always linked up to a PHP API I had built on an existing Web Application using RetroFit and Room

With Firebase, I don't have to write an API or most of the backend structure like Models, Repositories, ViewModels

- Mostly for a laugh though, animals are fun


Stack

Android / XML / Java / Material / Firebase
